## Running the search apps locally

1. Install `Aurelia CLI` and `npm`
2. Navigate to `/searchapps`
3. run `npm install`
4. run `au run --watch`
5. The the three demos are run at:

    http://localhost:9000/#simple
    
    http://localhost:9000/#facets
    
    http://localhost:9000/#location
